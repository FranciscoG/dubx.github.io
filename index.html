Install

You can install our extension on Chrome, Safari, Opera and Microsoft Edge. Firefox is now supported. You can also use out User.JS extension or the Chrome Store to install and our extension will auto-run in the browser whenver Dubtrack is visited.


User.JS Extension

Options

Our extensions offers a lot of options and addons for Dubtrack:

  • Autovote: Automatically clicks the like button when the song advances
  • Fullscreen: Covers the area of the window with the video and the chat.
  • Work Mode: Hides the chat and the video for use in the work place.
  • Split Chat: Differentiates the color of the incoming chat to make it easier to view.
  • Bug Report: Will open a dalouge to send a bug report to our Slack.
  • Feature Request: Will open a dialouge to send a feature request to our Slack.
  • Custom CSS: Will open a dialouge to store a custom CSS file.
  • We have a lot of upcoming options now that we have recieved the Dubtrack API.
